# Purpose
Here are my configuration files and some programs that I use on my MacBook

# Requirements
* MacOS

# Usage
I wrote a Makefile to handle all, the syntax is as follows:

## Getting Help
    $ make help

## Installing my .bashrc
    $ make bashrc

## Installing my .zshrc
    $ make zshrc

## Installing my .git-prompt.sh
    $ make gitprompt

## Installing my .tmux.conf
    $ make tmux

## Installing my vim theme
    $ make vim

## Installing my kubectl completion
    $ make kubectl

## Installing Terraform
    $ make terraform

## Installing Go
    $ make go

## Installing jq
    $ make jq

## Installing Pip
    $ make pip

## Installing awscli
    $ make awscli

## Installing eksctl
    $ make eksctl

## Installing all
    $ make all
